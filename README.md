# Supply Chain Decision Engine

## Overview

The Supply Chain Decision Engine is a Python-based decision support tool inspired by preventive quality planning and supply chain optimization processes used in industrial environments.

The project evaluates operational criteria and recommends whether a Standard PMQ Process or a Modified PMQ Process should be applied.

## Business Problem

Industrial organizations must constantly balance:

- Risk
- Supplier availability
- Time-to-market
- Cost efficiency

Using the wrong process can increase lead times, costs, and operational complexity.

This project provides a structured decision-making approach based on predefined business rules.

## Current Features

- Risk evaluation
- Supplier evaluation
- Process recommendation
- Modular architecture
- Separation of business logic and user interface
